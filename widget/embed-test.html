Kaal Chatbot Embed Test

This is a test page to verify the Kaal Chatbot widget embedding works correctly.

Note: Make sure the widget is built with npm run build:widget and the server is running on port 4000.

The chat widget should appear in the bottom-right corner. Click the mascot icon to open it.

Test Scenarios:

  • Widget launcher button visible
  • Widget opens on click
  • Persona selection works
  • Messages can be sent
  • Markdown rendering works (try asking about features)
  • Context-aware quick options appear
